# 🏠 Housing Data Analysis Dashboard

## 📌 Overview
This project analyzes housing data using Python and Streamlit. It provides interactive visualizations and insights based on feature correlations.

## 🎯 Objectives
- Perform data preprocessing
- Analyze correlations between features
- Visualize data using heatmaps and pairplots
- Generate automated insights

## 🛠️ Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Streamlit

## 📂 Project Structure
project/
│
├── train.csv
├── app.py
├── requirements.txt
└── README.md

## 🚀 How to Run

1. Install dependencies:

pip install -r requirements.txt


2. Run the app:

streamlit run app.py


## 📊 Features
- Interactive feature selection
- Correlation heatmap
- Pairwise relationships
- Automatic insights generation

## 🧠 Key Insights
- Strong correlation between living area and price
- Overall quality significantly impacts price
- Some features show weak or negative relationships

## 📌 Conclusion
The project demonstrates how data visualization and correlation analysis can help understand relationships in real-world datasets.
